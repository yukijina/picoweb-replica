Introducing PICO.

A stupidly simple & blazing fast, flat file CMS.

Making the web easy.

Stupidly Simple

Pico makes creating and maintaining a website
as simple as editing text tiles.

Blazing Fast

Pico is sericously lightweight and doesn't use a
database, making it super fast.

No Database

Pico is a "flat file" CMS, meaning no database woe's.
no MySQL queries, nothing.

Markdown Formatting

Edit your website in your favorite text editor using
simple Markdown formatting

Twing Template

Pico uses the Twing templating engine, for powerful
and flexible thems

Open Sourse

Pico is completely free and open source, released under the MIT licence
